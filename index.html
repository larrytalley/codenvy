Codenvy Proof-of-Concept

Codenvy is a multiuser development environment based on Eclipse Che. This Codenvy proof-of-concept will implement a Vaadin-based web user interface that consumes services deployed via Dropwizard, using Google Protocol Buffers and Apache ActiveMQ.

Why Codenvy

The problem we are trying to solve is that our developers support multiple projects or tasks, projects are implemented with a variety of technology stacks, and we spend too much time managing our development environments. We are hoping to demonstrate Codenvy customizable development environments per project or technology stack, composable, with a fully configured IDE customized for each project and sharable via a URL. Ideally a developer could be assigned to a project or task, given the IDE URL, and they could be committing code within an hour.

Proof-of-Concept Environments

In order to demonstrate that development environments can be customized per project or technology stack or task, we are demonstrating three independent environments:

1) Vaadin UI development

2) Services development

3) ActiveMQ configuration
